Insulation Batting Tools

Small browser-based tools for exploring and reproducing the insulation batting pattern from the original AutoCAD LISP routine. Use the path-based page to generate batting along arbitrary paths, or the horizontal-fill page to fill a rectangle with the repeating batt motif.

Disclaimer: these browser versions are inspired by the original LISP, but they are not a 1:1 conversion. The algorithm was tweaked for browser drawing, smoothing, clipping, interactive editing, and especially corner handling / corner rounding behavior.

Path Generator

Interactive Paper.js version of the batting tool. Draw or choose a path, then generate insulation curls with thickness, justification, and corner-handling controls.

Open insul-batting-paper.html Path generator spec

Horizontal Fill

Fills a resizable rectangle with the same batt pattern along a horizontal axis. Includes spacing controls, upper/lower gaps, clipping, and mouse-based resize handles.
